## What goes here

A growing, opinionated catalog of plugins that extend what assistants and coding agents can do: new tools, lifecycle hooks, model routing, memory and context backends, MCP integrations, and bundled skills.

A **plugin** is a host-facing extension package. It installs into an agent runtime and registers new capabilities (tools, hooks, providers, commands). This repo tracks the ones worth knowing about and the runtimes they work with.

## Suggest a plugin

Open an issue or a PR with:

- **Name** of the plugin
- **Runtime(s)** it works with (Claude Code, Codex, OpenClaw, Hermes, Cursor, etc.)
- **One line** on what it adds (tools, hooks, routing, memory, MCP, etc.)
- **Link** to the repo or docs

Keep entries neutral and concrete. No marketing copy.
